# Checklist de Qualidade — Content Squad

Use esta checklist antes de entregar qualquer conteudo ao usuario.

---

## CARROSSEL

### Hook (Slide 1)
- [ ] Maximo 8 palavras no titulo principal
- [ ] Cria curiosidade, promete beneficio ou faz afirmacao forte
- [ ] Funciona sem contexto (qualquer pessoa entende ao ver)
- [ ] 3 opcoes de hook foram entregues

### Estrutura
- [ ] Cada slide tem apenas uma ideia central
- [ ] Ha tensao no final de cada slide (usuario quer ver o proximo)
- [ ] Numero de slides adequado ao formato (lista: 6-11, vendas: 8-12, etc.)
- [ ] Ultimo slide tem CTA especifico e claro

### Copy
- [ ] Maximo 40 palavras por slide (ideal: 15-25)
- [ ] Tom de voz alinhado com BRAND_CONTEXT.md
- [ ] Nenhuma palavra proibida presente
- [ ] Palavras-chave obrigatorias incluidas (quando aplicavel)

### Visual
- [ ] Prompt DALL-E entregue para cada slide
- [ ] Consistencia visual indicada (paleta, estilo, personagem)
- [ ] Alternativa Canva fornecida
- [ ] Formato de exportacao especificado (PNG 1080x1080 ou 1080x1350)

### Publicacao
- [ ] Caption pronto com hook nas primeiras 125 chars
- [ ] Hashtags em 3 categorias (grande/medio/nicho)
- [ ] Horario recomendado incluido
- [ ] Checklist pre-publicacao entregue ao usuario

---

## REEL / SCRIPT

### Abertura
- [ ] Hook nos primeiros 3 segundos
- [ ] Sem logo ou intro no inicio
- [ ] Funciona sem audio (texto na tela presente)
- [ ] 3 opcoes de hook entregues

### Roteiro
- [ ] Cada cena especifica: visual + fala + texto na tela
- [ ] Duracao indicada por cena
- [ ] Total de duracao dentro do alvo (15s, 30s ou 60s)
- [ ] CTA no final (especifico)

### Producao
- [ ] Notas de filmagem incluidas (iluminacao, enquadramento)
- [ ] Notas de edicao incluidas (ritmo, musica, efeitos)
- [ ] Sugestao de thumbnail

### Publicacao
- [ ] Caption adaptado para Reels (mais curto, CTA rapido)
- [ ] Hashtags (3-8 para Reels)

---

## CAPTION / POST

### Hook
- [ ] Primeiras 125 chars sao o hook (antes do "ver mais")
- [ ] Cria curiosidade para continuar lendo
- [ ] Nao comeca com "Oi!" ou o nome da marca

### Corpo
- [ ] Paragrafos curtos (1-3 linhas)
- [ ] Linha em branco entre paragrafos
- [ ] Desenvolvimento do hook com valor real

### CTA
- [ ] Uma unica acao solicitada
- [ ] Especifico (nao "curta se gostou")
- [ ] Posicionado antes das hashtags

### Hashtags
- [ ] 1-3 grandes (100k+ posts)
- [ ] 3-5 medias (10k-100k)
- [ ] 3-5 nicho (<10k)
- [ ] Separadas do texto por linha em branco

---

## BRAND ALIGNMENT (todas as pecas)

- [ ] Tom de voz consistente com BRAND_CONTEXT.md
- [ ] Produto/servico posicionado corretamente
- [ ] Dores do publico endereçadas
- [ ] Palavras proibidas ausentes
- [ ] Nenhuma afirmacao que contradiz o posicionamento da marca

---

## NOTAS DE USO

Esta checklist e usada internamente pelos agentes do content-squad.
O content-chief e responsavel por validar antes da entrega final.
Se algum item falhar, o agente correspondente revisa antes de entregar.
